  • HTML5: البنية والترميز الدلالي
  • CSS3: التصميم والتصميم المتجاوب

أساسيات HTML

تعلم لبنات بناء صفحات الويب باستخدام HTML5. أتقن الترميز الدلالي والنماذج والوسائط المتعددة وإمكانية الوصول.

  • بنية ودلالات HTML5
  • النماذج والتحقق من صحة الإدخال
  • عناصر الوسائط المتعددة
  • أفضل ممارسات إمكانية الوصول

تصميم CSS

  • محددات وخصائص CSS3
  • تخطيط Flexbox و Grid
